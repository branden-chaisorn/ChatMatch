getTitleForUrl

fun getTitleForUrl(url: String): String

Overrides TitleRetriever.getTitleForUrl

Retrieves the page title for the given url

Note: If a IOException occurs, the string "badRequest" is returned as the title Note: If a SocketTimeoutException occurs, the string "socketTimeout" is returned as the title

Parameters

url - the given url

Exceptions

IOException - thrown if a bad URL is encountered, returns badURL as string for further error handling
